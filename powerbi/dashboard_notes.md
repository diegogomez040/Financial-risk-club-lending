# Power BI Dashboard Design Guide
## Lending Club Loan Performance & Risk Analysis (2007–2018)

This guide describes the recommended layout, data connections, visuals, and KPI cards for the interactive Power BI dashboard.

---

## Data Source Files

Import these 7 CSV files from `data/powerbi/` (generated by `notebooks/04_powerbi_export.ipynb`):

| File | Contents | Used on Page |
|------|----------|-------------|
| `01_portfolio_summary.csv` | Top-level KPIs | Overview |
| `02_defaults_by_grade.csv` | Risk & profit by grade A–G | Risk by Segment |
| `03_defaults_by_purpose.csv` | Risk & profit by purpose group | Risk by Segment |
| `04_defaults_by_year.csv` | Vintage trends 2007–2018 | Portfolio Trends |
| `05_defaults_by_fico.csv` | Risk by FICO score band | Risk by Segment |
| `06_defaults_by_term.csv` | Risk by 36/60-month term | Risk by Segment |
| `07_defaults_by_dti.csv` | Risk by DTI band | Risk by Segment |

**How to import:** Power BI Desktop → Get Data → Text/CSV → select each file → Load.

## Dashboard Pages

### Page 1 — Portfolio Overview

**Purpose:** Executive summary with top-level KPIs at a glance.

**KPI Cards (single row across the top):**
- Total Loans: `total_loans` from `01_portfolio_summary.csv`
- Total Funded: `total_funded_M` (format as "$X.X Billion")
- Overall Default Rate: `default_rate_pct` (format as "X.X%")
- Total Net Profit: `total_profit_M` (format as "$X.X Million")
- Average Interest Rate: `avg_int_rate_pct` (format as "X.X%")
- Average FICO Score: `avg_fico_score`

**Charts:**
- **Donut chart** — Defaulted vs Non-Defaulted loans (use `total_defaults` and `total_loans` from summary)
- **Bar chart** — Loan volume by grade (from `02_defaults_by_grade.csv`: `grade` vs `loans`)
- **Text box** — Brief project description and data source note

---

### Page 2 — Risk by Segment

**Purpose:** Deep-dive into default rate drivers across key risk dimensions.

**Layout: 2×3 grid of charts**

**Top row:**
1. **Bar chart** — Default Rate by Grade (`grade` vs `default_rate_pct`) — color-coded from green (A) to red (G)
2. **Bar chart** — Default Rate by FICO Band (`fico_band` vs `default_rate_pct`, sorted by `sort_order`)
3. **Bar chart** — Default Rate by DTI Band (`dti_band` vs `default_rate_pct`, sorted by `sort_order`)

**Bottom row:**
4. **Clustered bar chart** — Default Rate & Portfolio Share by Purpose (`purpose_group` vs `default_rate_pct` and `pct_of_portfolio`)
5. **Bar chart** — Default Rate by Term (`term_label` vs `default_rate_pct`) — 36 months vs 60 months
6. **Table / Matrix** — Grade breakdown with all metrics: loans, defaults, default rate %, avg interest rate, profit

**Slicers (filter panel on the left):**
- Grade (multi-select checkboxes)
- Purpose Group (dropdown)
- Loan Term (toggle: 36 / 60)

---

### Page 3 — Portfolio Trends

**Purpose:** Time-series analysis of loan volume, default rates, and average loan amounts over 2007–2018.

**Charts (stacked vertically):**
1. **Line & Clustered Column** combo chart:
   - Bars: `loans` (loan volume by year)
   - Line: `default_rate_pct` (default rate trend)
   - X-axis: `issue_year`
   - Dual Y-axes (volume on left, rate on right)

2. **Line chart** — Average Loan Amount Over Time (`issue_year` vs `avg_loan_amnt`)
   - Format Y-axis as currency ($)

3. **Line chart** — Average Interest Rate Over Time (`issue_year` vs `avg_int_rate`)

**Slicer:**
- Year range slider (2007–2018)

---

### Page 4 — Profitability Analysis

**Purpose:** Compare profit/loss by credit grade and loan purpose.

**Charts:**
1. **Bar chart (horizontal)** — Profit by Purpose Group (`purpose_group` vs `profit`)
   - Color positive bars green, negative bars red using conditional formatting
   - Add data labels showing profit/loss in $M

2. **Bar chart** — Profit by Credit Grade (`grade` vs `profit`)
   - Same conditional formatting

3. **Scatter chart** — Default Rate vs Profit by Grade
   - X-axis: `default_rate_pct`
   - Y-axis: `profit`
   - Bubble size: `loans` (portfolio share)
   - Color: `grade`
   - This shows the risk-return trade-off visually

4. **KPI card** — Total portfolio net profit (from `01_portfolio_summary.csv`)

## Formatting Guidelines

### Color Palette
- **Low risk / positive:** `#2ca02c` (green)
- **Medium risk:** `#ff7f0e` (orange)
- **High risk / negative:** `#d62728` (red)
- **Neutral / volume:** `#1f77b4` (blue)
- **Background:** White or light gray (`#f8f9fa`)

### Typography
- Title font: Segoe UI Bold, 14–16pt
- Chart labels: Segoe UI, 10–11pt
- KPI numbers: Segoe UI Bold, 18–24pt

### General
- Enable tooltips on all charts showing exact values on hover
- Add page-level titles in the top-left of each page
- Use consistent axis formatting: percentages as "X.X%", currency as "$X,XXX"
- Add a data source footer: "Source: Lending Club 2007–2018 | Kaggle Dataset"

---

## Recommended Interactions

- **Grade slicer** on Page 2 should cross-filter all charts on the same page
- **Year slicer** on Page 3 should filter all trend charts
- **Drill-through:** Right-click a grade on the Overview page → drill to the Risk by Segment page filtered to that grade

---

## Tips for Power BI Import

1. **Open Power BI Desktop** → Home → Get Data → Text/CSV
2. Import each of the 7 CSV files as separate tables
3. No relationships needed between tables (each is independently aggregated)
4. If you want to add a full raw data view, you can also import `cleaned_data.csv` directly (1.37M rows, 90 MB) — Power BI handles this well with DirectQuery or Import mode
5. Use **New Measure** (DAX) if you want dynamic calculated fields, e.g.:
   ```dax
   Default Rate = DIVIDE(SUM(table[defaults]), SUM(table[loans]), 0) * 100
   ```
